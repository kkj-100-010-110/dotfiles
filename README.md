# .dotfiles

**A minimal setup focused on C/C++ development on Debian-based systems.
These dotfiles are tailored for personal use and updated regularly.**

- **OS** : Debian/Ubuntu

## Components
    - .bashrc
    - .vimrc
    - .gitconfig
    - setup.sh
    - install.sh

## How to run
    1. git clone repo
        git clone https://github.com/kkj-100-010-110/dotfiles.git ~/.dotfiles
    2. install packages
        cd ~/.dotfiles
        chmod +x setup.sh
        ./setup.sh
    3. run install.sh
        chmod +x install.sh
        ./install.sh
